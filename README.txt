Установка:
1) Установить Python
	Перейдите по ссылке:
	https://www.python.org/downloads/
	И нажмите на кнопку:
	Download Python 3.X.X (на момент написания Python 3.6.2)
2) Установить Pygame
	А устанавливается пайгейм так:
	Запускаете командную строку(Пуск->Выполнить->cmd) и введите:
	python3 -m pip install pygame --user
	Если выдаст ошибку, то введите:
	python -m pip install pygame --user

Запуск:
Запускать игру через файл RunGame.py

Управление:
Передвижение персонажа WASD
Стрелять на стрелочки
